# KỶ NGUYÊN HEALTH CARE

Trang web giới thiệu dịch vụ chăm sóc sức khỏe cá nhân tại bệnh viện và nhà riêng.

## Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (Vanilla)

## Cài đặt và chạy

1. Clone repository về máy
2. Mở file `index.html` trong trình duyệt

## GitHub Pages

Trang web được host trên GitHub Pages tại: `https://[username].github.io/kynguyenhealthcare`
